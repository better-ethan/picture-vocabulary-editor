### TODO

This is a very simple todo list web built with TRPC and React Router. It is not meant to be a production ready app, but rather a simple example of how to use TRPC and React Router together.

### Features

- Add a todo
- Delete a todo
- Mark a todo as completed
- List all todos

### Tech Stack

- Pnpm monorepo
- TypeScript
- Drizzle ORM
- PostgreSQL
- TRPC
- React Router
- Tailwind CSS
- Vite

### Getting Started

1. Clone the repository
2. Install dependencies
3. Start the app

```bash
docker compose up --build -d
```

4. Open your browser and navigate to `http://localhost:3000`
